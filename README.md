# Mi Portafolio Profesional

Portafolio personal construido con **Astro** + **Tailwind CSS** con un diseño moderno, responsivo y con excelente UX/UI.

## 🚀 Características

- ✅ Diseño totalmente responsivo (mobile, tablet, desktop)
- ✅ Animaciones suaves y transiciones fluidas
- ✅ Paleta de colores profesional (gradientes)
- ✅ Secciones: Hero, Sobre Mí, Habilidades, Proyectos, Contacto
- ✅ Formulario de contacto funcional
- ✅ Optimizado para SEO
- ✅ Rendimiento rápido (Astro es muy rápido)
- ✅ Accesibilidad (a11y)

## 🛠️ Tecnologías

- **Astro** - Framework meta moderno
- **Tailwind CSS** - Utilidades CSS
- **TypeScript** - Tipado estático
- **HTML5** & **CSS3**

## 📝 Instrucciones de uso

### Desarrollo local

```bash
npm install
npm run dev
```

Visita `http://localhost:3000` en tu navegador.

### Build para producción

```bash
npm run build
npm run preview
```

## 📂 Estructura del proyecto

```
src/
├── components/     # Componentes reutilizables
├── layouts/        # Layouts
├── pages/          # Páginas
└── styles/         # Estilos globales
```

## 🎨 Personalización

Para personalizar el portafolio:

1. Reemplaza "Carlos" con tu nombre en los componentes
2. Actualiza los datos de contacto en `Contact.astro`
3. Añade tus proyectos en `Projects.astro`
4. Modifica los colores en `tailwind.config.mjs`
5. Actualiza las habilidades en `Skills.astro`
